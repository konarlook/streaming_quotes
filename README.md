# Quote streaming

## Ручной сценарий проверки

Справочник тикеров сервера лежит в `assets/tickers.txt`:

```                                                                                                                                                                                                                                                                                                                                                                                                                                       
APL,GPN,WRD,GROW,FMR                                                                                                                                                                                                                                                                                                                                                                                                                      
```                                                                                                                                                                                                                                                                                                                                                                                                                                       

Клиент читает свой список тикеров тоже из файла. Создайте файл
подписки с 2–3 тикерами из справочника сервера, например `assets/client_tickers.txt`:

```                                                                                                                                                                                                                                                                                                                                                                                                                                       
APL,GPN                                                                                                                                                                                                                                                                                                                                                                                                                                   
```            

### Терминал А

```sh                                                                                                                                                                                                                                                                                                                                                                                                                                     
cargo run --bin server -- -f assets/tickers.txt                                                                                                                                                                                                                                                                                                                                                                                           
```                                                                                                                                                                                                                                                                                                                                                                                                                                       

По умолчанию сервер принимает команды на `127.0.0.1:7878` (TCP) и шлёт
котировки с `127.0.0.1:9000` (UDP-порт-источник). Адреса можно поменять
флагами `--server-addr` и `--udp-addr`.

### Терминал В

```sh
cargo run --bin client -- --udp-addr 127.0.0.1:9001 -f assets/client_tickers.txt
```

`--udp-addr` — адрес, на котором клиент принимает котировки. Он не должен
совпадать с `--udp-addr` сервера (`9000`): этот порт уже занят исходящим
сокетом сервера. TCP-адрес сервера по умолчанию `127.0.0.1:7878`
(флаг `--tcp-addr`).

### Что должно происходить

- В консоли клиента раз в секунду бегут котировки только по тикерам
  из файла подписки (`APL`, `GPN`), остальные (`WRD`, `GROW`, `FMR`) не приходят.
- Клиент шлёт `PING` по TCP раз в секунду. Если остановить клиент (
  Ctrl-C), сервер перестаёт получать пинги и при очередной чистке (каждые 5 с)
  удаляет подписку по таймауту - UDP-стрим для этого подписчика прекращается.
- Второй клиент поднимается независимо — другой UDP-порт и другой файл подписки:

  ```sh                                                                                                                                                                                                                                                                                                                                                                                                                                   
  cargo run --bin client -- --udp-addr 127.0.0.1:9002 -f assets/client_tickers_2.txt                                                                                                                                                                                                                                                                                                                                                      
  ```                                                                                                                                                                                                                                                                                                                                                                                                                                     

где `assets/client_tickers_2.txt` содержит, например, `WRD,GROW`.
Оба клиента получают каждый свой набор котировок; остановка одного не влияет на
другого.
